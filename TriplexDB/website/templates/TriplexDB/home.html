Welcome to TripLexicon, the database of RNA:DNA:DNA triple helix formation powered by TriplexAligner!

TripLexicon is a database holding predicted gene regulatory RNA:DNA:DNA triplexes. The triplexes were predicted using TriplexAligner. In its current form, TripLexicon holds predicted RNA:DNA:DNA triplexes between human and murine lncRNAs (as annotated in GENCODE v45 & GENCODE vM35, respectively) and human and mouse gene regulatory elements consisting of promoters (TSS -2500/+500) and regulatory elements from EpiRegio and ATAC sequencing data.

Explore the triplexicon queries

  • RNA Query: explore predicted triplexes by RNA gene or transcripts ( docs )
  • Target Query: explore predicted triplexes by DNA target genes ( docs )
  • Region Query: explore predicted triplexes by genomic regions in which DNA target elements reside ( docs )
